# 📈 Python and Statistics for Financial Analysis (HKUST)
Ce dépôt regroupe mes travaux réalisés dans le cadre de la certification de la Hong Kong University of Science and Technology. L'objectif est de démontrer la transition des outils traditionnels (Excel) vers une approche quantitative et automatisée avec Python pour l'analyse des marchés financiers.

🛠 Stack Technique
• Langage : Python 3.x

• Bibliothèques clés :
                       Pandas & NumPy : Manipulation de séries temporelles et calculs vectoriels.
                       Matplotlib & Seaborn : Visualisation de données boursières et distributions.
                       Statsmodels : Modélisation statistique et régressions linéaires.
                       SciPy : Tests d'hypothèses et statistiques avancées.

📂 Structure du Portfolio
Module 1 : Visualisation et Manipulation de Données

 Projet : Mise en place d'une stratégie de Trend Following (suivi de tendance).
 Compétences : Importation via API/CSV, nettoyage de DataFrames, calcul de moyennes mobiles (MA10/MA50) et génération de signaux d'achat/vente.
 
Module 2 : Variables Aléatoires et Distributions
Projet : Modélisation du risque boursier.
Compétences : Analyse de la fréquence des rendements journaliers logarithmiques, loi normale et calcul de probabilités de perte/gain.

Module 3 : Échantillonnage et Inférence Statistique
• Projet : Tests d'hypothèses sur les rendements.
• Compétences : Calcul d'intervalles de confiance pour la moyenne des prix et validation statistique des claims de rendement (p-value).

Module 4 : Modèles de Régression Linéaire
• Projet final : Prédiction de l'ETF S&P 500 (SPY).
• Détails : Construction d'un modèle de régression multiple utilisant les indices globaux (LSE, N225, HSI) pour prédire les variations du SPY.
• Évaluation : Analyse des résidus, calcul du Ratio de Sharpe et du Drawdown Maximum.

📊 Indicateurs de Performance Maîtrisés
J'utilise des métriques standardisées pour évaluer la robustesse des modèles financiers :
• Ratio de Sharpe : Mesure du rendement ajusté au risque.
• Drawdown Maximum (MDD) : Évaluation du risque de perte maximale de sommet à creux.
• R² (Coefficient de détermination) : Capacité du modèle à expliquer la variance du prix.
